Voici notre SAE24 : localisation d’un objet sonore en temps réel.

Le projet consiste à simuler et afficher sur une cartographie web la position d’un objet détecté par trois microphones, dans une salle de 8 × 8 mètres. L’ensemble repose sur un MQTT simulé, une base MySQL (PHPmyadmin) et une interface web dynamique.
