Aspect technique

Le fonctionnement du projet est entièrement automatisé par les scripts bash, php et crontab :

  • Base de données MySQL de XAMPP pour le stockage des mesures et des positions estimées.
  • Mosquitto exécuté dans un conteneur Docker pour la transmission en temps réel des mesures.
  • Scripts Bash pour automatiser :
    • La simulation du déplacement (génération d’un point mobile et de mesures pour chaque micro).
    • La captation des mesures via MQTT et l’insertion en base SQL.
    • Le calcul de la position estimée toutes les 2 secondes selon les amplitudes reçues.
  • Mise en place du crontab pour relancer automatiquement après le redémarrage.
  • Cartographie :
    • Affichage sur une page PHP des dernières positions calculées.
    • Cartographie à jour toutes les 2 s, avec tracé de la trajectoire et repères des microphones.
  • Travail en équipe piloté par GitHub pour le code, la documentation et la coordination entre les membres de l'équipe.
